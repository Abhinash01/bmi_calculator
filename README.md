# BMI-Calculator <br>
This project is a simple and interactive Body Mass Index (BMI) Calculator built using HTML, CSS, and JavaScript. It allows users to input their height and weight to calculate their BMI and get an instant health status (Underweight, Normal, Overweight, or Obese) based on standard BMI categories. The main goal of this project is to strengthen JavaScript skills by working with user input, conditionals, DOM manipulation
